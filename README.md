# Level Devil
I made this game to be a challenging and creative game. The game should make the player nervous and want to finish the game in any way. I used Finn Sprite because I think that he is suitable for this game and I liked this character (from Adventure Time). This game consists of three levels. I challenge you to finish the game or even reach level 3. If you reached, tell me. I tried to make this game hard by making my friends play it. Just two of my friends could reach level 3 but not finish the game. I wish you luck; you need it. The game should be challenging with the traps that you don't know where it is and that you can not trust anything, even the floor could be deceiving. <p>

### How to win

You can win only by passing level 1 and then level 2 and 3. After level three, you will see a message "You win" and then you fall in the hole beside.
